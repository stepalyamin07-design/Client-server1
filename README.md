
# Клиент-серверное приложение 

Приложение для обмена сообщениями и файлами между несколькими компьютерами в одной локальной сети (класс, офис).

---

## Как это работает

- Сервер — один компьютер в сети. Хранит историю чата, принимает и отдаёт файлы, обрабатывает команды.
- Клиенты — все остальные компьютеры. Подключаются к серверу, отправляют сообщения, переключают комнаты, обмениваются файлами и вызывают серверные функции.

---

## Возможности

- Текстовый чат по комнатам
- Передача файлов (клиент -> сервер, сервер -> клиент)
- Подсчёт пробелов в текстовом файле на сервере
- Сохранение истории чата на сервере (/save) и её просмотр (/view)
- Групповая рассылка сообщений всем клиентам (/broad)
- Многопоточная работа с несколькими клиентами
- Автоматическая генерация имён: User1, User2, ...

---

## Команды клиента

| Команда | Действие |
|---------|----------|
| Любой текст | Отправить сообщение всем в текущей комнате |
| /room <название> | Переключиться в другую комнату (по умолчанию main) |
| /spaces <файл> | Подсчитать пробелы в файле на сервере (файл рядом с server.exe) |
| /sendfile <полный_путь> | Отправить файл на сервер -> сохранится как received_<имя> |
| /getfile <имя> | Скачать файл с сервера в папку клиента |
| /save <комната> | Сохранить историю комнаты в chat_history_<комната>.txt (на сервере) |
| /view <комната> | Показать сохранённую историю комнаты |
| /broad <сообщение> | Разослать сообщение всем клиентам (независимо от комнаты) |
| /help | Показать справку |
| /exit | Отключиться от сервера |

---

## Создаваемые файлы

На сервере:
- received_* — принятые от клиентов файлы
- chat_history_<комната>.txt — сохранённые истории чатов

На клиенте:
- Файлы, скачанные через /getfile (появляются в папке с client.exe)

---

## Настройка сети

По умолчанию клиент подключается к 127.0.0.1 (тот же компьютер).

Чтобы подключиться к серверу в локальной сети — измените в client.cpp:

const char* SERVER_IP = "127.0.0.1";

Например, если сервер имеет IP 192.168.1.10:

const char* SERVER_IP = "192.168.1.10";

Как узнать IP сервера в Windows:
- Откройте командную строку (cmd)
- Введите команду ipconfig
- Смотрите строку IPv4-адрес

---

## Запуск

1. Запустите server.exe на компьютере-сервере
2. Запустите client.exe на клиентских компьютерах
3. Вводите команды или просто печатайте сообщения

---

## Используемые технологии

- C++
- Winsock2 (Windows Sockets)
- TCP
- Многопоточность (<thread>)

---

## Примечание

- Сервер должен быть запущен до клиентов
- Все клиенты должны иметь сетевой доступ к серверу
- Файлы для /spaces должны лежать рядом с server.exe
